# 分支说明

***

此分支基于master，用于服务器端开发

有问题请提交Issue

# Web端相关

***

## 环境需求

* php5.3及以上

* PDO扩展和MySQLi扩展

* MySQL

## 安装方法

Client：

	cd ClientWeb
	cd install
	php install.php

Server：

	cd ServerWeb
	cd install
	php install.php

## 涉及目录

* ServerWeb 服务器端网页

* ClientWeb 客户端网页
